## 六、换IP之后要注意什么

几个容易被忽略的后续操作：

- **更新DNS解析**：如果你有绑定域名，记得把解析记录改成新IP，不然访问域名还是指向旧地址
- **更新本地配置**：SSH客户端、代理软件里写死了旧IP的，都要改
- **等DNS生效**：DNS缓存需要一点时间，改完之后不是立刻全球都生效，等十几分钟再测试
- **数据不会丢**：换IP不影响服务器上的数据和服务，只是网络地址变了
